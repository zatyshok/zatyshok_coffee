ІНСТРУКЦІЯ З ЗАПУСКУ ВЕБПЛАТФОРМИ "ЗАТИШОК"
============================================

Для того, щоб запустити розроблений дипломний проект на вашому комп'ютері, вам знадобиться лише встановлений Python (бажано версії 3.10+).

КРОК 1: 
Переконайтеся, що у вас встановлено Python. 
Якщо його немає - завантажте з офіційного сайту: https://www.python.org/downloads/
(ВАЖЛИВО: під час установки поставте галочку "Add Python to PATH").

КРОК 2:
Двічі клікніть по файлу `start.bat`, який знаходиться в цій папці.
Цей файл автоматично:
- Встановить всі необхідні бібліотеки для роботи сервера та штучного інтелекту.
- Запустить локальний сервер.
- Самостійно відкриє ваш браузер із готовим сайтом.

КРОК 3:
Поки ви користуєтеся сайтом, не закривайте чорне вікно консолі (воно працює як ваш локальний сервер).
Коли закінчите тестування - просто закрийте чорне вікно, і сервер зупиниться.

ДОДАТКОВО:
- База даних вже містить початкове меню (товари) та тестових користувачів.
- Ключ до Google Gemini API вже прописано у файлі config.py, чатбот повністю налаштований і готовий до тестування.

Успіхів!)
