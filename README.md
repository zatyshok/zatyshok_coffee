# Кав'ярня «Затишок» — Дипломний проект

Проект адаптивного вебінтерфейсу з AI-чатботом для кав'ярні.

## Як запустити проект

1. **Встановіть Python** (якщо ще немає):
   Завантажте з офіційного сайту: https://www.python.org/downloads/
   Під час встановлення обов'язково поставте галочку **"Add Python to PATH"**.

2. **Встановіть бібліотеки**:
   Відкрийте термінал (або командний рядок CMD) у папці з проектом і виконайте команду:
   ```bash
   pip install flask flask-sqlalchemy werkzeug google-generativeai requests
   ```

3. **Налаштуйте AI-чатбота (Gemini API)**:
   - Перейдіть на сайт: https://aistudio.google.com/app/apikey
   - Увійдіть під своїм Google-акаунтом.
   - Натисніть кнопку **"Create API key"**.
   - Скопіюйте отриманий ключ.
   - Відкрийте файл `config.py` у папці проекту за допомогою будь-якого текстового редактора (наприклад, Блокнота).
   - Знайдіть рядок `GEMINI_API_KEY = ...` і вставте свій ключ замість `YOUR_GEMINI_API_KEY_HERE`.

4. **Запустіть сервер**:
   У терміналі в папці проекту виконайте команду:
   ```bash
   python app.py
   ```

5. **Відкрийте сайт**:
   Перейдіть у браузері за адресою: http://127.0.0.1:5000/
